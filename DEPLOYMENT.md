## 🎯 How to Use This Repository

### Option 1: Clone to Local Machine

```bash
# Clone the repository
git clone https://github.com/suryaaiclaude-boop/rent.git
cd rent

# Install dependencies
npm install

# Run the application
npm run dev
```

Then open: **http://localhost:3000**

### Option 2: Download ZIP from GitHub

1. Go to https://github.com/suryaaiclaude-boop/rent
2. Click the green "Code" button
3. Click "Download ZIP"
4. Extract and follow steps above

## 🌐 Deployment Options

### Deploy to Vercel (Frontend)
```bash
npm install -g vercel
vercel
```

### Deploy to Heroku (Full-Stack)
```bash
heroku create rent-management-app
git push heroku main
```

### Deploy to Railway (Full-Stack)
1. Visit https://railway.app
2. Connect GitHub repository
3. Deploy automatically

### Deploy to Render (Full-Stack)
1. Visit https://render.com
2. New Web Service
3. Connect GitHub repository
4. Build command: `npm install && npm run build`
5. Start command: `npm run server`
