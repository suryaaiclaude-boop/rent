## 🎯 Features You'll Get

### 🏠 Property Management
- Add, edit, and delete rental properties
- Track property types (Apartment, House, Studio, Condo)
- Monitor rent amounts
- View tenant count per property
- See total revenue per property

### 👥 Tenant Management
- Store tenant contact information (name, email, phone)
- Assign tenants to properties
- Track lease start and end dates
- Edit and remove tenant records

### 💰 Payment Tracking
- Record rent payments
- Multiple payment methods (Bank Transfer, Check, Cash, Credit Card, PayPal)
- Track payment status (Completed, Pending, Failed)
- Add optional notes
- View complete payment history

### 📊 Dashboard
- Total properties count
- Total tenants count
- Total revenue calculation
- Recent payments table
